DSABot Documentation

Getting Started

Welcome to DSABot! This bot is designed to help Discord communities learn and practice Data Structures and Algorithms together.

DSABot provides various features including practice problems, resource sharing, and community engagement tools to make learning DSA more interactive and fun.

Running Locally

To run DSABot locally, you'll need Python 3.8+ and pip installed on your system. Follow the setup instructions below to get started with your own copy of the bot.

Make sure you have a Discord Developer account and have created a bot application in the Discord Developer Portal before proceeding.

Setup

Follow these steps to set up the bot:


# Clone the repository
git clone https://github.com/kashsuks/DSABot

# Install dependencies
pip install -r requirements.txt

#Or if you're using a VENV (Virtual Environment)
python3 -m venv {venv_name}
source {venv_name}/bin/activate

# Configure environment variables in the `.env` file
DISCORD_TOKEN = 'your discord token'
KATTIS_USERNAME = 'your kattis username'
KATTIS_PASSWORD = 'your kattis passowrd'
DATABASE_URL = 'your postgresql database url'

Commands

Resources Command

This command gives the user the resources that I use to practice Data Structures and Algorithms


/resources

The current output for /resources command are the following links:

  1. https://cp-algorithms.com/index.html

  3. https://leetcode.com/
  4. https://codeforces.com/
  5. https://open.kattis.com/

Random Problem

Generates practice problems based on difficulty level and topic.


/random-problem

How Does This Command Work?

By using the OpenKattis api alongside the credentials of a registered Kattis account, you are able to use the kt.suggest method which suggests problems for you account. This means that if you have access to a new Kattis account without any problems solved, this should give a pseudorandom output.

Random Algorithm

Gives a random algorithm for you to learn.


/random-algorithm

How Does This Command Work?

As of now, the Random Algorithm command uses a pre written/hard-coded array of algorithms of which, Algorithms are picked randomly and outputted.

Leaderboard Command

Displays server-wide rankings based on codeforces rating.


/leaderboard

How Does This Command Work?

The command takes the Codeforces Rating of all the registered users and ranks them, from highest rating to lowest. After this process, the command processes the first 10 ratings that are then returned. Below this output(a red embed) is the users own rank. It shows their name and their rank on this leaderboard.

Github Repository Link

Displays the Github repository link of the Discord Bot.


/repo

You can access the repository here: DSABot Repository

APIs

DSABot integrates with various APIs to provide a comprehensive learning experience. The bot uses LeetCode's API for problem fetching and verification, and maintains its own API for user progress tracking.
