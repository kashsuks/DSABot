DSABot Documentation

Getting Started

Welcome to DSABot! This bot is designed to help Discord communities learn and practice Data Structures and Algorithms together.

DSABot provides various features including practice problems, resource sharing, and community engagement tools to make learning DSA more interactive and fun.

Running Locally

To run DSABot locally, you'll need Python 3.8+ and pip installed on your system. Follow the setup instructions below to get started with your own instance of the bot.

Make sure you have a Discord Developer account and have created a bot application in the Discord Developer Portal before proceeding.

Setup

Follow these steps to set up the bot:


# Clone the repository
git clone https://github.com/yourusername/DSABot

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
import os
os.environ['DISCORD_TOKEN'] = 'your-token-here'
os.environ['PREFIX'] = '!'  # Optional command prefix

Commands

Resources Command

The resources command provides curated learning materials and references.


async def handle_resources(interaction):
    resources = await get_resources_by_topic()
    embed = discord.Embed(
        title="DSA Learning Resources",
        color=discord.Color.blue()
    )
    await interaction.response.send_message(embed=embed)

Practice Command

Generates practice problems based on difficulty level and topic.


async def handle_practice(interaction, difficulty: str, topic: str = None):
    problem = await generate_problem(difficulty, topic)
    embed = create_problem_embed(problem)
    await interaction.response.send_message(embed=embed)

Timer Command

Sets up a countdown timer for practice sessions.


async def handle_timer(interaction, minutes: int):
    timer = PracticeTimer(minutes)
    await interaction.response.send_message(
        f"Timer started for {minutes} minutes!"
    )
    await timer.start()

Leaderboard Command

Displays server-wide rankings based on problem-solving activity.


async def handle_leaderboard(interaction):
    rankings = await get_server_rankings(interaction.guild_id)
    embed = create_leaderboard_embed(rankings)
    await interaction.response.send_message(embed=embed)

APIs

DSABot integrates with various APIs to provide a comprehensive learning experience. The bot uses LeetCode's API for problem fetching and verification, and maintains its own API for user progress tracking.
